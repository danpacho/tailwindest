# `mergeProps`

## Briefly

Core style generator `function` to handle toggle conditions.

## 1. Type definition

```ts
declare const toggle: (
    booleanCondition: boolean,
    {
        falsy,
        truthy,
        base,
    }: {
        truthy: StyleType
        falsy: StyleType
        base?: StyleType
    }
) => string
```

## 2. Spec

### Usage

```ts
toggle(booleanCondition, {
    base: baseStyle,
    truthy: truthyStyle,
    falsy: falsyStyle,
})
```

### Parameter: `booleanCondition`

-   type: `boolean`
-   usage: style condition

### Parameter: style, `base`

-   type: `Tailwindest`
-   usage: common style for `true` and `false` condition

### Parameter: style, `truthy`

-   type: `Tailwindest`
-   usage: if condition is `true`, it will applied.

### Parameter: style,`falsy`

-   type: `Tailwindest`
-   usage: if condition is `false`, it will applied.

## 3. Returns

**className string**
